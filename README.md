# Streamero Arcade

A gaming app built with Next.js, TypeScript, and Tailwind CSS featuring mini games and leaderboards with a streaming-style design.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - React components
- `lib/` - Utility functions and game data
- `components/ui/` - Reusable UI components

## Features

- Game browsing and details pages
- Global leaderboard
- Streaming-style design system
- Responsive layout
